The Elements of Style (Japanese Translation)
========

Build instruction
--------

```shell
$ git clone https://github.com/hisashim/elements-of-style-1918-ja/
$ cd elements-of-style-1918-ja
$ bundle install
$ rake
```

Requirements
--------

Prerequisites:

  * Git
  * Ruby

Build requirements:

  * [Ruby](https://www.ruby-lang.org/)
  * [Bundler](https://bundler.io/)
  * [Rake](https://ruby.github.io/rake/)
  * [YAMultilingualMarkdown](https://github.com/hisashim/ya_multilingual_markdown)
